# 🎬 Stranger Things Experience - Landing Page

Uma landing page interativa inspirada no universo de *Stranger Things*, desenvolvida com foco em animações modernas e experiência imersiva.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* GSAP (ScrollTrigger, ScrollSmoother, SplitText)

## ✨ Funcionalidades

* 🎥 Animações suaves com GSAP
* 🔄 Efeitos de scroll interativos (ScrollTrigger)
* 🌊 Smooth scroll com ScrollSmoother
* 🔤 Animação de textos (SplitText)
* ⏳ Preloader animado
* 📱 Design responsivo
* 🎨 Layout moderno inspirado na Netflix
